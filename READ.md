# Universitetssystem i C#


Dette prosjektet er laget som en skoleoppgave. Programmet simulerer et universitetssystem.

## Funksjoner:

- Opprette kurs

- Melde student på kurs

- Søke etter kurs

- Registrere bøker

- Låne og returnere bøker

## Teknologi:

- Språk: C#

- Type: Console Application

- Fokus: Objektorientert programmering
